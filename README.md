# Hemanth Kumar Portfolio

A modern, interactive, and fully responsive Developer Portfolio website built to showcase skills, experience, and projects with a premium touch. This portfolio features advanced animations, 3D particle effects, and a seamless user experience.

## ‚ú® Features

- **Modern & Responsive Design**: Built with a mobile-first approach, ensuring a stunning look on all devices (Desktop, Tablet, Mobile).
- **Core Technologies**: Powered by **Next.js 14**, **React 18**, and **Tailwind CSS**.
- **Advanced Animations**: Smooth page transitions and element animations using **Framer Motion**.
- **Interactive UI**:
  - **3D Particle Effects**: Integrated using `tsparticles` for a dynamic background depth.
  - **Infinite Logo Loop**: Custom-built seamless scrolling marquee for tech skills.
  - **Laser-Glow Buttons**: Unique button interactions with rotating gradient borders.
- **Projects Showcase**: Interactive slider using **Swiper.js** to browse through recent work.
- **Contact Form**: Functional contact form powered by **EmailJS** for direct messaging.
- **Performance Optimized**: Fast loading speeds with Next.js optimization and lightweight assets.

## üöÄ Getting Started

To run this project locally, follow these steps:

### Prerequisites

Make sure you have **Node.js** (v16.8.0 or newer) installed.

### Installation

1.  **Clone the repository**:
    ```bash
    git clone https://github.com/HemanthKumar52/Portfolio-main.git
    cd modern-portfolio
    ```

2.  **Install dependencies**:
    ```bash
    npm install
    # or
    yarn install
    ```

3.  **Run the development server**:
    ```bash
    npm run dev
    ```

4.  **Open locally**:
    Visit [http://localhost:3000](http://localhost:3000) in your browser.
